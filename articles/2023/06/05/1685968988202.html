Spring Bean 条件装配注解
Spring 注解 场景说明 起始版本

Spring 4.0 之后，@Profile 基于 @Conditional(ProfileCondition.class) 实现

 1class ProfileCondition implements Condition {
 2	@Override
 3	public boolean matches(ConditionContext context, AnnotatedTypeMetadata metadata) {
 4		MultiValueMap<String, Object> attrs = metadata.getAllAnnotationAttributes(Profile.class.getName());
 5		if (attrs != null) {
 6			for (Object value : attrs.get("value")) {
 7				if (context.getEnvironment().acceptsProfiles(Profiles.of((String[]) value))) {
 8					return true;
 9				}
10			}
11			return false; // 返回 false 时，Bean不会被加载
12		}
13		return true;
14	}
15}

  • 自定义 BeanDefinitionParser 实现：XML 元素与 BeanDefinition 解析

    继承 AbstractSingleBeanDefinitionParser 并重写 getBeanClass 和 doParse() 方法

     1public class CustomUserBeanDefinitionParser extends AbstractSingleBeanDefinitionParser {
 2
 3    @Override
 4    protected Class<?> getBeanClass(Element element) {
 5        return User.class;
 6    }
 7
 8    @Override
 9    protected void doParse(Element element, ParserContext parserContext, BeanDefinitionBuilder builder) {
10        serPropertyValue("id", element, builder);
11        serPropertyValue("name", element, builder);
12        serPropertyValue("city", element, builder);
13    }
14
15    private void serPropertyValue(String attributeName, Element element, BeanDefinitionBuilder builder) {
16        String attributeValue = element.getAttribute(attributeName);
17        if (StringUtils.hasText(attributeValue)) {
18            builder.addPropertyValue(attributeName, attributeValue);
19        }
20    }
21}

FAQ

什么是 outter beans ？

如下 xml 文件中的 beans 就是 dependency-lookup-context.xml 中的 outter beans

 1<?xml version="1.0" encoding="UTF-8"?>
 2<beans xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
 3       xmlns="http://www.springframework.org/schema/beans"
 4       xsi:schemaLocation="http://www.springframework.org/schema/beans
 5       http://www.springframework.org/schema/beans/spring-beans.xsd">
 6
 7    <!--复用context.xml-->
 8    <import resource="dependency-lookup-context.xml"/>
 9
10    <bean id="userRepository" class="thinking.in.spring.ioc.overview.repository.UserRepository"
11          autowire="byType">
12    </bean>
13</beans>
